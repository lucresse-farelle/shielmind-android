# shielmind-android
Application Android de contrôle parental intelligent basée sur l'IA  embarquée (TensorFlow Lite). Filtrage adaptatif de contenus textuels  en temps réel avec notifications push



## Équipe
- Membre 1 - Modèle IA
- Membre 2 - Android Core
- Membre 3 - Android UI
- Membre 4 - DevOps / Tests
## Technologies
- Kotlin 1.9+
- Jetpack Compose
- TensorFlow Lite
- Firebase FCM
- Room Database
## Build
./gradlew assembleDebug
