# Guide d'Installation

## Prérequis

- Android Studio Arctic Fox ou plus récent
- SDK Android API 24 ou plus
- Kotlin 1.9.0+

## Installation

1. Clonez le dépôt:
```bash
git clone https://github.com/shielmind/shielmind-android.git
```

2. Ouvrez le projet dans Android Studio

3. Laissez Gradle faire la synchronisation automatique

4. Compilez et exécutez l'application

## Configuration

Assurez-vous que les dépendances suivantes sont installées:
- Android SDK 34
- Kotlin plugin
