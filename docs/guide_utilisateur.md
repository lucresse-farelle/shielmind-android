# Guide Utilisateur

## Qu'est-ce que ShieldMind ?

ShieldMind est une application Android de sécurité intelligente qui utilise le machine learning pour détecter et prévenir les menaces.

## Fonctionnalités principales

### 1. Surveillance d'accessibilité
L'application utilise le service d'accessibilité pour surveiller l'activité du système.

### 2. Analyse avec IA
Utilise TensorFlow Lite pour classifier et identifier les menaces potentielles.

### 3. Notifications
Reçoit des alertes en temps réel en cas de menace détectée.

## Comment utiliser

1. Lancez l'application
2. Accordez les permissions d'accessibilité
3. L'application commence à surveiller automatiquement
